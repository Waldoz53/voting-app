# React + SQLite Voting App

Straight forward voting app with React for the front end, and SQLite3, Express, Node for the backend.

Set up and seed the server in node
```
node setup-db.js
```
```
node seed-db.js
```

Then run the server
```
node server.js
```

And run the React app, if in dev mode
```
npm run dev
```
